PMD (Propeller Material Design) Admin

Download file_download Preview arrow_forward

PMD (Propeller Material Design) is a responsive admin dashboard template created mainly for admin and backend applications(CMS, CRM, Custom Admin Application, Admin Dashboard). PMD template is powered with Bootstrap Framework and Material Design standards. It comes with a huge collection of reusable UI components and integrated jQuery plugins. PMD template works on all major web browsers, tablets and phones.

Enjoy using PMD template and stay tuned for amazing updates!

Feature Highlights

  • done Built with material design guidelines and Bootstrap framework.
  • done All pages are HTML5 validated and neatly coded.
  • done Fully Responsive design, supported for desktops, laptops, smartphones and tablets.
  • done Complete User Interface elements and components.
  • done Sample pages included such as Profile, Messages, Listview, Login etc.
