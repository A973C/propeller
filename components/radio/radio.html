Radio

Use radio buttons for exclusive selection if you think that the user needs to see all available options side-by-side. Add .pmd-radio to create a propeller customized radio. You can also add .pmd-radio-pmd-ripple-effect to provide a ripple effect to the radio. Inside this div you need to also add a span element with .pmd-radio-label.

Inline Radio - Use .radio-inline on a series of radio boxes for controls that appear on the same line.

Configuration Options

.pmd-textfield-floating-label-completed Class added dynamically after completion of floating animation of the label of input Optional

.has-success To create a success input field Optional

.checkbox-inline Add this class to create inline checkboxes Optional
.pmd-radio To create propeller customized radiobutton Optional

.togglebutton To create toggle switch Optional
